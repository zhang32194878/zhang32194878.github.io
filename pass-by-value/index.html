地址传递

  • java地址传递

    输出结果 : 5

地址传递 同样的将地址传递给change的形参,c传递了整个指针,但是java传递的仅仅是地址

举个栗子（￣︶￣）↗ : c和java各牵着条狗,若你想牵这条狗,c的做法是将狗链直接给你,java的做法是在找一条狗链牵到狗脖子上把新的那条给你

  • java并没有像 *x = 3 这种操作(java封装掉了)
